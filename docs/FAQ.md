# FAQ

## Is this a public release repo?

No. This is the private staged FT repo formed during Phase 3 and Phase 4.

## Does the repo prove Timescale equivalence?

No. The Timescale path remains `INCONCLUSIVE` and is deferred to a later rerun.

## Why is there Rust code if quick verify may report `python-fallback`?

The Rust-core source is preserved in `core/`, but the staged repo currently ships a Python fallback bridge by default. This phase did not add a packaged Rust build/install workflow.

## Are the historical runbooks and raw logs current instructions?

No. Some of them are preserved only as historical evidence and still carry old workstation paths. Use the repo front door and audit docs for current interpretation.
