# Support

Use support requests for installation or interpretation questions, not for security disclosures.

## Include With Any Support Request

- exact command you ran
- current branch and commit
- Python version
- whether `rust_bridge` reported `python-fallback` or a compiled backend
- the file or proof artifact you were using when the issue appeared

## Routing

- security issue: see [SECURITY.md](SECURITY.md)
- evidence dispute or claim disagreement: include the contradicting command output and proof path
- repo-boundary/doc issue: open a scoped change with the correction
